Correctif TOTEM

Lorsqu'on fait un export de Revit vers ifc, les Pset_TOTEM sont "mal exportés" de sorte qu'on ne puisse pas utiliser l'application TOTEM (en gros). Il est demandé que le nom des objets auxquels sont associés le Pset_TOTEM soit identique à la valeur de la propriété "Type_TOTEM" de ce même Pset. Pour ce faire, un script Python a été rédigé (disponible ici) qui utilise la bibliothèque IfcOpenShell.

Le correctif ne s'applique qu'aux fichiers *.ifc qui possède un Pset TOTEM. Un autre format de fichier ou un fichier sans Pset TOTEM renverra une erreur.
